Repositorio con los códigos de las simulaciones de la ecuación de Schrodinger dependiente del tiempo mediante el método numérico de las diferencias finitas.

Requisitos:
  1. Python 3.x
  2. Librerías: NumPy, SciPy, Matplotlib
  3. Entorno recomendado: Visual Studio Code o Jupyter Notebook

Ejecución:
Cada simulación está implementada en un archivo .ipynb. Puede ejecutarse abriendo el notebook en VS Code o Jupyter y ejecutando las celdas en orden.
