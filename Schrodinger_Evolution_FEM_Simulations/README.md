Repositorio con los códigos de las simulaciones de la ecuación de Schrodinger dependiente del tiempo mediante el método numérico de los elementos finitos. 

Requisitos: 
  1) FreeFem++ instalado en el sistema.
  2) Ejecución desde la terminal

Ejecución:
Cada simulación puede ejecutarse desde la terminal con: FreeFem++ nombre_del_script.edp
